Application Processing

The Shiny application prompts the user to input a car cylinder number. Car cylinders only come in 4, 6 or 8. Once the user selects a car cylinder, the app will do the following:

    1. Return the number you entered
    2. Return the Minimun Displacement
    3. Return the Mean Displacement
    4. Return the Maximum Displacement
